Excel Sales Performance Dashboard
Project Overview

This project analyzes the regional sales performance of ABC Electronics using Microsoft Excel. The objective was to clean the raw sales data, perform business analysis, and develop an interactive dashboard to help management understand sales trends and make informed decisions.

Objectives
Clean and preprocess sales data
Analyze sales by region and product category
Create Pivot Tables for summarization
Design an interactive Excel dashboard
Generate business insights from sales data

Key Insights
East region generated the highest sales.
Clothing was the highest-selling product category.
Total Sales exceeded ₹1 Crore.
Interactive slicers enable dynamic business analysis.
